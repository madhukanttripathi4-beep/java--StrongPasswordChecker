# PasswordStrengthChecker

This program checks how strong a password is based on simple rules.  
It counts uppercase letters, lowercase letters, digits and unique characters, gives a rating out of 10, and shows a level such as Very Weak, Moderate, Good or Excellent.  
If the password is not strong enough, the user must enter it again.

## Rules

The password must have:

- atleast 12 characters
- atleast 4 Upper Case Characters
- atleast 4 Lower Case Characters
- atleast 3 Digits
- atleast 1 Unique Character (like @,#,$,%)
